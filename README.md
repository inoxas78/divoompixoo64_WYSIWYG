# divoompixoo64_WYSIWYG
Pixoo Designer OS : Éditeur WYSIWYG et moteur de rendu pour Divoom Pixoo 64 sous Home Assistant. Dépassez l'API native via son backend Pyscript : incrustation de texte dynamique sur GIFs animés (MultiGif), dithering 3-3-2 matériel, polices pixel-perfect et rastérisation d'icônes MDI en direct. L'impossible devient réalité !
